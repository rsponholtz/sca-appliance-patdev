SCA Pattern Libraries for Python

Libraries have been written to help write SCA Patterns. The following libraries are available for python.

  • Core.py - Base libraries
  • HAE.py - High Availability Extension cluster specific libraries
  • MPIO.py - High Availability Extension cluster specific libraries
  • oes.py - Open Enterprise Server specific libraries
  • suma.py - SUSE Manger specific libraries
  • SUSE.py - SUSE specific libraries
  • Xen.py - Xen specific libraries
